/api/catalog

This method returns the list of products as a json array.

/api/catalog/<category>

This method finds the products that belong to category and return the products as a json.

/api/catalog/id/<id>

This method returns the product with the specific id.

/api/catalog/cheapest

This method get the cheapests product and return it as a json.

/api/categories

This method give a list of unique categories as a json array of strings.

/api/test

This method adds and removes products, get the length of the array of products, print the list and some elements.
